# 🌌 Play The God

**Play The God** is an interactive web-based crafting game where you start with fundamental particles and combine them to create the entire universe. 

It starts with just Quarks, Electrons, Photons, and pure Nothingness. By figuring out the right physics and space combinations, you can craft elements, planets, black holes, and eventually win the game by creating the Universe.

## ✨ Features
* **Physics & Space Logic:** Over a dozen logical combinations taking you from the quantum scale to the cosmic scale.
* **Smart Hint System:** Stuck? The game checks your inventory and gives you a custom hint for the exact next item you need to discover.
* **Modern UI:** A clean, space-themed design using "Glassmorphism" (frosted glass effects), glowing buttons, and CSS star animations.
* **Music Player:** A built-in, floating audio player to set the deep-space mood while you play.
* **Server-Side Checking:** All game logic and combinations are verified by a Python backend, making it secure and scalable.

## 🛠️ Built With
* **Backend:** Python & Flask
* **Frontend:** HTML5, Vanilla JavaScript, and raw CSS
* **Hosting:** Deployed live on Render using Gunicorn
